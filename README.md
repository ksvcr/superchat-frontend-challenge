# superchat-frontend-challenge

Demo: https://superchat-frontend-challenge-phi.vercel.app/

## Stack

React, Typescript, Next.js, Graphql, Tailwind CS

## Development

1. Prepare env variable
```
  # Create .env.local with
  
  GITHUB_AUTH_TOKEN=***
```

2. Install dependencies
```
  npm i
```

3. Run

```
  npm run dev
```

4. Run storybook

```
  npm run storybook
```

5. Run tests

```
  npm run tests
```
